# Repository for my Matlab PID Projects
This repository contains the livescripts for my functions and projects involving PIDs in Matlab

The phySim live script is a function that runs a basic 1d physics simulation of a massless "drone" flying up and down in the air

The PID's Demo is a live script that shows how a the three different parts of a PID affect the motion and stability of a craft

The Manually Tuned PID is just a cleaned up live script containing a full PID with the Kp,Ki, and Kd coefficients having to been manually modified

The PID runner live script is a function version of the Manually Generated PID 

The Randomly Generated PID is a livescript that runs a bunch of PID simulations with different Kp,Ki, and Kd coefficients and giving back the best one.
